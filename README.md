# Bus-Tracking-Application

Description: an android based application built with 3 modules user, driver and admin. The application is develped on android studio and with java and realtime firebase technologies. The application allows users to track the bus and avoid wasting their time by taking alternative options available if bus is not arraiving on its time.

System flow: 
1. user: Register/Login -> track the available buses
2. driver: login -> turn on GPS location.
3. admin: Register/login -> add driver or add bus -> save the data.

Status: all core features for all the modules works fully, only the bus location details at users end are not accessible due to logical errors
